# COS314_Assignment2
## Iterated Local Search (ILS) for the 0/1 Knapsack Problem
### How to run and compile
1. Compile
  javac ILS.java
2. Run
  java ILS
OR
Import the jar file to your IDE and compile and run it there.

### User Input

At runtime, the program will prompt for:
1. Seed value
  Any integer value for reproducibility
2. Path to the folder containing instance files
  Example:
    C:\Users\john\OneDrive\Documents\Knapsack Instances

Notes:
+ Do not include quotation marks in the path
+ Ensure the folder path is correct
+ The program supports folder paths with spaces
